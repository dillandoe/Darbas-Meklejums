Šis projekts ir vienkārša, moderna HTML un CSS tīmekļa lapa, kurā apkopotas populārākās darba meklēšanas vietnes Latvijā.
Lapas mērķis ir palīdzēt cilvēkiem ātrāk un ērtāk atrast darbu, piedāvājot tiešās saites uz uzticamiem resursiem, piemēram, NVA, CV.lv, CV-Online.lv un Visidarbi.lv.

Dizains ir izstrādāts tā, lai būtu pievilcīgs, viegli uztverams un motivējošs — ar ikoniņām, krāsainām kartītēm un vieglu fonu ar smaidiņiem, kas simbolizē pozitīvu ceļu uz jaunu darbu.
